Redirecting

Through redirection you can direct the input and output of a command to and from other files and programs, and chain commands together in a pipeline.

Input and output (I/O) redirection

Command What it does How to use it
echo

The command accepts strings as standard input, and echoes the string back to the terminal as standard output.
$ echo "Hello"

Standard input, standard output, and standard error

standard input

Abbreviated as stdin, is information inputted into the terminal through the keyboard or input device.
$

standard output

Abbreviated as stdout, is the information outputted after a process is run.
$

standard error

Abbreviated as stderr, is an error message outputted by a failed process.
$

Redirection reroutes standard input, standard output, and standard error to or from a different location
